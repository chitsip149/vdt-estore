# E-Store - Online Bookstore Website

This is a simple online bookstore website where users can browse books, add them to the cart, place orders, and make online payments.

## Technologies Used

### Backend
- Java 17  
- Spring Boot  
- Spring Security (JWT)  
- JPA & Hibernate  
- MySQL  
- Flyway  
- Stripe API (Online Payment)

### Frontend
- ReactJS  
- Axios  
- React Router
